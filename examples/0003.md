## Day 3 Examples

Create a table with a primary key using a number of 5

```sql
CREATE TABLE DEC21_DEMO2(
id number(5) primary key,
name varchar2(20));
```

```sql
insert into DEC21_DEMO2 values(101,'jack');
insert into DEC21_DEMO2 values(102,'jerry');
```

```sql
DROP TABLE DEC21_DEMO2;
DELETE FROM DEC21_DEMO2;
TRUNCATE TABLE DEC21_DEMO2;
```
